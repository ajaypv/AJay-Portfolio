Features

  • ▹

  • ▹

    Open Dots employs collaborative-filtering machine learning algorithms to match users based on shared interests.

  • ▹

2. Aerox

Aerox is a project designed to streamline printing by automating the process and providing users with real-time updates on queue wait times. With its mobile app, users can conveniently monitor their print job's progress and retrieve it as soon as it's ready. Additionally, Aerox offers virtual queuing functionality for food ordering at cafeterias and eateries, allowing customers to browse menus, place orders, and track their status directly from their mobile devices.

Features

  • ▹

    Aerox Android App enables smooth PDF document uploads and printing, developed using JAVA, Android Studio, AWS, and Firebase.

  • ▹

    Seamless in-app UPI payments facilitated through Retrofit and REST APIs, with Firebase and express.js managing backend data efficiently

  • ▹

    Real-time communication via Socket.io ensures a user-friendly experience, complemented by a JavaFX-based desktop application for vendors, offering print monitoring and revenue analytics within the Aerox ecosystem

Project Aim: Optimizing Printing and Dining Experiences

Aerox revolutionizes printing efficiency with real-time queue management. It also offers virtual queuing for food ordering at cafeterias and eateries, enabling mobile browsing, order placement, and status tracking. Prioritizing convenience, sustainability, and cost-effectiveness for businesses.
